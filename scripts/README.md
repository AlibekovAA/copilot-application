# Test Scripts

Скрипты для тестирования API.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Тест API

```bash
python test_api.py
```

Скрипт выполнит 5 тестов:

1. Health Check
2. Общий чат (general domain)
3. Юридический домен (legal)
4. Маркетинговый домен (marketing)
5. Финансовый домен (finance)

### Результаты

Результаты сохраняются в папке `scripts/`:

- `scripts/test_results_YYYY-MM-DD_HH-MM-SS.json` - JSON формат
- `scripts/test_results_YYYY-MM-DD_HH-MM-SS.txt` - текстовый формат

## Примеры

### Базовый тест

```bash
cd scripts
python test_api.py
```

### С виртуальным окружением

```bash
cd scripts
python -m venv venv
venv\Scripts\activate  # Windows
# или
source venv/bin/activate  # Linux/Mac

pip install -r requirements.txt
python test_api.py
```
